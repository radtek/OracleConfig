# OracleConfig

Create commands to import and export oracle.

## REQUIRE
- .NET CORE 2.2
- Newtonsoft.Json
	
## HELP

-	Put config client Json into folder client_json.
	Output commands oracle into folder client_command.

	```
	$ dotnet OracleConfig.dll client
	```
